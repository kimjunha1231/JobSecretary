# Antigravity Instructions (Pointer)

이 프로젝트는 오픈AI의 Harness Engineering 워크플로우를 따릅니다. 
따라서 상세한 기술 지침, 코딩 컨벤션, UI/UX 디자인 시스템, 에러 헨들링 규칙 등은 이곳에 보관하지 않습니다.

작업을 시작하거나 아키텍처 룰을 파악하려면 무조건 **루트 디렉토리의 `AGENTS.md`**를 먼저 읽으세요.
이후 `AGENTS.md`의 지시에 따라 `docs/harness/` 폴더 내부의 문서들을 단계별로 참고하여 작업하시면 됩니다.

**[CRITICAL INSTRUCTION - 기계적 검증 강제 조항]**
너(Antigravity 에이전트)는 코드를 수정하거나 새로운 기능을 추가한 직후, 사용자에게 태스크 완료를 보고하기 전에 **반드시 터미널에서 `npm run harness:verify`를 실행해야 합니다.** 
이 명령어는 FSD 아키텍처 룰 위반 여부(Lint), 타입스크립트 에러, 그리고 유닛 테스트를 한 번에 검증합니다. 만약 이 명령어가 실패한다면, 스스로 에러를 디버깅하고 코드를 수정한 뒤 다시 명령어를 돌려 완전히 통과할 때까지 피드백 루프를 반복하세요. 절대 이 검증 명령어를 통과하기 전에 "작업이 완료되었다"고 보고하지 마세요.
