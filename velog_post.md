# 구글 Antigravity와 Gemini로 20시간 만에 만든 AI 취업 비서

> "취업 준비, 너무 복잡하고 힘들지 않나요? AI가 내 비서가 되어준다면 어떨까요?"

안녕하세요! 이번에 **Google의 최신 AI 에이전트인 Antigravity와 Gemini**의 도움을 받아, 단 20시간 만에 **AI 통합 채용 관리 플랫폼 'JobSecretary'**를 개발하게 되어 그 과정을 공유하고자 합니다.

## 🚀 프로젝트 소개: JobSecretary

**JobSecretary**는 취업 준비생들의 가장 큰 고민인 '자기소개서 작성'과 '면접 준비'를 AI로 해결해주는 서비스입니다.

### 주요 기능
1.  **AI 자기소개서 초안 작성**: 키워드만 입력하면 Gemini가 합격 자소서 데이터를 기반으로 초안을 작성해줍니다.
2.  **AI 면접 질문 생성**: 내 자소서를 분석하여 나올 법한 예상 면접 질문을 뽑아줍니다.
3.  **채용 공고 칸반 보드**: 지원 현황을 한눈에 파악하고 드래그 앤 드롭으로 관리할 수 있습니다.
4.  **태그 기반 아카이빙**: 작성한 자소서를 태그별로 정리하여 필요할 때 바로 찾아볼 수 있습니다.

## 🛠️ 기술 스택 (Tech Stack)

짧은 시간 안에 완성도 높은 프로덕트를 만들기 위해 생산성이 높은 최신 스택을 선택했습니다.

*   **Framework**: Next.js 15 (App Router)
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS, Framer Motion (애니메이션)
*   **State Management**: Zustand, React Context API
*   **Backend/DB**: Supabase (Auth, Database)
*   **AI**: Google Gemini API (`@google/generative-ai`)
*   **Deployment**: Vercel

## 🤖 AI와의 협업 과정 (Development Process)

이번 프로젝트의 핵심은 **'AI와의 페어 프로그래밍'**이었습니다. 단순히 코드를 짜달라고 하는 것을 넘어, 기획부터 배포까지 전 과정에서 AI를 활용했습니다.

### 1. 기획 및 설계 단계
막연한 아이디어를 구체화하는 데 AI가 큰 역할을 했습니다.
*   *"취업 준비생에게 가장 필요한 기능이 뭘까?"* 라는 질문에 대해 AI는 '자소서 관리'와 '면접 대비'를 제안했고, 이를 바탕으로 데이터베이스 스키마와 API 구조를 설계했습니다.

### 2. 핵심 기능 구현 (feat. Gemini)
가장 시간이 많이 걸리는 로직 구현을 AI에게 맡겨 시간을 단축했습니다.
*   **AI 자소서 생성**: 프롬프트 엔지니어링을 통해 양질의 자소서를 뽑아내는 로직을 구현했습니다.
*   **칸반 보드**: `@dnd-kit` 라이브러리를 사용한 복잡한 드래그 앤 드롭 로직을 AI가 거의 완벽하게 스캐폴딩해주어, 저는 스타일링과 엣지 케이스 처리에만 집중할 수 있었습니다.

### 3. SEO 최적화 및 배포
개발 막바지에는 검색 노출을 위한 SEO 작업도 AI와 함께했습니다.
*   `robots.txt`, `sitemap.xml` 생성부터 `Next.js`의 Metadata API를 활용한 동적 메타 태그 설정까지, AI가 제안한 코드를 적용하여 검색 엔진 친화적인 사이트를 구축했습니다.

## 💡 느낀 점 (Retrospective)

### "개발자의 역할이 바뀌고 있다"
예전 같으면 1주일은 족히 걸렸을 프로젝트를 20시간 만에 끝내면서, 개발자의 역할이 **'코더(Coder)'에서 '아키텍트(Architect) & 디렉터(Director)'**로 변화하고 있음을 체감했습니다.

AI는 단순 반복 작업과 복잡한 로직 구현을 대신해주었고, 저는 **사용자 경험(UX) 설계, 코드 품질 관리, 그리고 '어떤 문제를 해결할 것인가'**에 더 집중할 수 있었습니다.

### 앞으로의 계획
*   **RAG(검색 증강 생성) 도입**: 사용자의 과거 경험 데이터를 벡터 DB에 저장하여, 더 개인화된 자소서를 써주도록 고도화할 예정입니다.
*   **모의 면접 기능**: 텍스트를 넘어 음성으로 대화하며 면접 연습을 할 수 있는 기능을 추가하려 합니다.
